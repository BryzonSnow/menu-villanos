# Menú Digital - Los Villanos

Aplicación web desarrollada para gestionar y visualizar el menú digital del restaurante "Los Villanos". Este proyecto permite a los usuarios consultar el catálogo de productos y precios actualizados a través de una interfaz dinámica.

## Características Principales

* Visualización dinámica del menú y catálogo de productos.
* Organización de los productos por categorías.
* Sistema ligero y rápido gracias al uso de SQLite.
* Interfaz responsiva y adaptable a dispositivos móviles.

## Tecnologías Utilizadas

* **Backend:** Python 3, Flask
* **Base de Datos:** SQLite
* **Frontend:** HTML5, CSS3 (Ajustar si usaste algún framework CSS o JavaScript)

## Requisitos Previos

Para ejecutar este proyecto en un entorno local, es necesario contar con:
* Python 3.8 o superior
* pip (gestor de paquetes de Python)
